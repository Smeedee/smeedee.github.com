IIS Troubleshooting

The server part of Smeedee is hosted by IIS, and if it is not correctly configured, Smeedee won't work. You may see symptoms like no Admin Tasks, never-ending loading of slideshow configuration or just no Smeedee at all! Here is a list of things you can check to verify your IIS configuration.

App Pool .NET version

Make sure that the application pool that the smeedee application runs in is set to use .NET 4.0.
